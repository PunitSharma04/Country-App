# 🌍 Countries Explorer

A modern React application that allows users to explore detailed information about countries around the world using the REST Countries API.

---

## 🚀 Live Demo

https://countryapireactapp.netlify.app

---

## 📌 Features

* 🔍 Search countries by name
* 🌎 Filter countries by region
* 📄 Detailed country information page
* 🏳️ Display country flags
* 🌐 Border countries navigation
* 🌙 Light / Dark mode toggle
* ⚡ Shimmer loading effect
* 🔙 Back navigation

---

## 📡 API Used

This project uses the REST Countries API to fetch real-time country data.

**Base URL:**

```
https://restcountries.com/v3.1/
```

---

## 🛠️ Tech Stack

* React.js
* React Router
* CSS (Custom + CSS Variables)
* REST API
